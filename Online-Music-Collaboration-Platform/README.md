# Online Music Collaboration Platform

A Java-based web application using JSP, Servlets, and JDBC.

## Features
- User Registration & Login
- Upload Music Files
- View All Uploaded Music
- Basic Collaboration Module

## Technologies Used
- JSP
- Servlets
- JDBC (MySQL)
- HTML/CSS

## Setup
1. Import project into Eclipse/VS Code.
2. Create MySQL database using `database.sql`.
3. Update DB credentials in `DBConnection.java`.
4. Run on Apache Tomcat Server.
